## 🧭 新手指南

**通过本指南，你可以在几分钟内快速上手`KiviBot`。**

> [!WARNING]
>
> 本框架并不推荐`刚注册的小号`使用，很大概率直接封号。

账号使用等问题见：[常见问题](/docs/problem)

## 🥚 快速上手

`KiviBot` 框架得益于 `Node.js` 的跨平台特性，可以在多个平台部署。

请根据你系统的版本，选择相应的教程：

import { Cards } from "nextra/components";
import { WindowsIcon, LinuxIcon, AppleIcon, AndroidIcon } from "@components/icons";

<Cards>
  <Cards.Card
    icon={<WindowsIcon />}
    title="Windows"
    href="/docs/start/windows"
  />
  <Cards.Card 
    icon={<LinuxIcon />} 
    title="Linux" 
    href="/docs/start/linux" 
  />
  <Cards.Card
    icon={<AppleIcon />}
    title="macOS"
    href="/docs/start/macos"
  />
  <Cards.Card
    icon={<AndroidIcon />}
    title="Android (Termux)"
    href="/docs/start/android"
  />
</Cards>

当然，这支持本地和云端。

如果比较爱折腾，且家里有网络更推荐部署`本地服务器`，

运存需求不高只需要 200mb 足够= =

> 如果你想要购买云服务器，推荐新用户优惠：
>
> 腾讯云：[轻量应用服务器](https://curl.qcloud.com/1DmnBzQv)
>
> 阿里云：[轻量应用服务器](https://www.aliyun.com/minisite/goods?userCode=syvtrib3)
