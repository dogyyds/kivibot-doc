## 🧐 框架介绍

![img](/kivibot-logo.png)
`🤖 KiviBot`是一款使用 [Typescript](https://www.typescriptlang.org/) 编写的一款 **轻量** **优雅** **跨平台** QQ 机器人框架。

- 框架的核心 QQ 协议基于 [icqq](https://icqq.pages.dev/) ，在此表示感谢。
- 本项目初衷在于**提高群活跃氛围**、**方便群管理**，仅供个人娱乐、学习交流使用。
- **不得**将本项目用于**任何非法用途**，若因用户个人原因而造成的一切不良后果**与框架作者无关**。

框架安装与使用详情，请移步[新手指南](/docs/start)。

## 🫨 Why KiviBot

- **📦 轻量**：无需运行 UI，内存占用相当低。
- **⚡️ 高效**：框架依赖少，执行效率高。
- **😎 优雅**：框架与插件均使用 `TypeScript` 编写，即写即用，支持热重载。
- **🖥️ 跨平台**：Windows, Linux, Mac, Android 等都能运行。
- **🥚 注重体验**：使用 QQ 消息 执行操作、启用或重载插件。
- **🚲 极速开发**：学习门槛低，只需几行 `Typescript` 代码就能编写插件。

## 🍃 插件示例

```typescript
import { definePlugin } from "kivibot";

export default definePlugin({
  name: "群复读",
  version: "1.0.0",
  async setup(ctx) {
    ctx.handle("message.group", (e) => {
      // 快速回复消息
      e.reply(e.raw_message);
    });
  },
});
```

> 你现在并不需要理解代码意思，你只需要知道编写插件很简单，这里只有 ctx.handle()构成的三行代码是用于实现群复读功能的，其他均为固定模版。

**没错只是几行代码，就实现了群聊复读的功能 (它会复读收到的每一条消息)。**

插件使用详细 见 [🧩 插件文档](/docs/plugins)。

插件开发详细 见 [🐣 开发文档](/docs/plugins)。
